AI Ebook Translator - User Manual

Welcome to the AI Ebook Translator. This application allows you to automatically translate E-Books (in .epub format) into German using modern AI technology.

Quick Start

1. Installation

Ensure all dependencies are installed:

pip install PyQt6 openai ebooklib beautifulsoup4 odfpy pyinstaller

2. Launching the App

You can run the program directly via Python:

python3 gui_app.py

Usage

Step 1: Configuration

On the first run, you need to configure your AI settings in the "AI Configuration" section:

  • API Key: Enter your API key here (e.g., from OpenAI or Moonshot AI).
  • Base URL: The API endpoint. Default is Moonshot AI (https://api.moonshot.ai/v1).
  • Model Name: The AI model to use (e.g., kimi-k2.5).
  • Workers: Number of files to translate in parallel. A value of 3 is usually optimal.

Settings are saved automatically.

Step 2: Selecting Files

  1. Click "Browse..." next to "E-Book (.epub)" to select your book.
  2. (Optional) Select a "Target Folder". If left empty, a new folder will be created in the book's directory.

Step 3: Start Translation

Click the green button Start Translation.

The program will automatically:

  1. Extract: Split the E-Book into chapters.
  2. Translate: Translate each chapter using AI.
  3. Merge: Combine translated texts into a new document (.odt).

Monitoring & Troubleshooting

  • Log Window: Shows detailed real-time activity.
  • Progress Bar: Displays overall completion status.
  • Stop: You can cancel the process at any time.
Resuming: If a translation fails or is stopped, restarting it with the same target folder will skip already translated files, saving you time and API costs.
