AI Ebook Translator - Benutzerhandbuch

Willkommen beim AI Ebook Translator. Diese Anwendung ermöglicht es Ihnen, E-Books (im .epub-Format) mithilfe moderner KI automatisch ins Deutsche zu übersetzen.

Schnellstart

1. Installation

Stellen Sie sicher, dass alle Abhängigkeiten installiert sind:

pip install PyQt6 openai ebooklib beautifulsoup4 odfpy pyinstaller

2. Programm starten

Sie können das Programm direkt über Python starten:

python3 gui_app.py

Bedienung

Schritt 1: Konfiguration

Beim ersten Start müssen Sie Ihre KI-Einstellungen im Bereich "AI Konfiguration" vornehmen:

  • API Key: Ihr API-Schlüssel (z.B. von OpenAI oder Moonshot AI).
  • Base URL: Die API-Adresse. Standard ist Moonshot AI (https://api.moonshot.ai/v1).
  • Model Name: Das KI-Modell (z.B. kimi-k2.5).
  • Parallele Worker: Anzahl gleichzeitiger Übersetzungen. 3 ist meist optimal.

Die Einstellungen werden automatisch gespeichert.

Schritt 2: Dateien auswählen

  1. Klicken Sie bei "E-Book (.epub)" auf "Durchsuchen...".
  2. (Optional) Wählen Sie unter "Zielordner" einen Speicherort.

Schritt 3: Übersetzung starten

Klicken Sie auf den grünen Button Übersetzung Starten.

Das Programm führt automatisch folgende Schritte aus:

  1. Extraktion: Zerlegt das Buch in Kapitel.
  2. Übersetzung: Übersetzt jedes Kapitel per KI.
  3. Zusammenfügen: Erstellt ein neues Dokument (.odt).

Überwachung & Fehlerbehebung

  • Verlauf: Zeigt alle Aktionen in Echtzeit an.
  • Fortschritt: Der blaue Balken zeigt den Gesamtstatus.
  • Abbruch: Sie können den Vorgang jederzeit stoppen.
Wiederaufnahme: Falls die Übersetzung fehlschlägt oder gestoppt wird, starten Sie sie einfach erneut im selben Zielordner. Das Programm überspringt bereits übersetzte Dateien automatisch.
